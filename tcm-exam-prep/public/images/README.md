# 药材图片和穴位图片

## 图片来源（无需翻墙）

### 药材图片 → `herbs/` 目录
**推荐来源：香港浸会大学中药材图像数据库**
https://libproject.hkbu.edu.hk/was40/search?channelid=1288&lang=ch

收录 420+ 种常用中药材高清图像，免费开放。按药材名搜索后右键保存图片。

**文件命名**：用药材中文名（如 `麻黄.jpg`、`桂枝.jpg`、`柴胡.jpg`）

### 穴位图片 → `acupoints/` 目录
**推荐来源：WHO标准经穴定位图**
或使用系统内置SVG解剖定位图（无需额外添加）

**文件命名**：用穴位代码（如 `LI4.jpg`、`ST36.jpg`、`SP6.jpg`）

### 支持格式
.jpg / .png / .webp

### 示例：快速获取药材图片
1. 打开 https://libproject.hkbu.edu.hk/was40/search?channelid=1288&lang=ch
2. 搜索 "麻黄"
3. 点击结果查看大图
4. 右键 → 图片另存为 → 保存到 `herbs/麻黄.jpg`
